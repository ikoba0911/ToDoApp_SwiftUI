# ToDoApp_SwiftUI
 
This is a simple todo application developed with SwiftUI.

# Usage

1. Clone this repository
```
$ git clone https://github.com/Ayamegu/ToDoApp_SwiftUI.git
```

2. Go into the repository
```
$ cd ToDoApp_SwiftUI
```

3. open application
```
$ open ToDoApp_SwiftUI.xcodeproj
```
